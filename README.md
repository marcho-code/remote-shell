## Tiny-Shell

  

> 帮助你解决镜像加速问题，如果有帮助到你，不妨点一个Star :star: 吧~~

  

### :triangular_flag_on_post: 支持的操作系统

![](./icons/debian.png)![](./icons/ubuntu.png)![](./icons/centos.png)

  

### tiny-shell 功能概览

  

- 快速切换、配置系统镜像源

- 配置Docker镜像源，快速安装Docker环境

- 快速配置PiP国内加速

- 快速配置Maven国内加速

  

### :zap: 快速开始

  

```bash

curl  -s  https://tiny-shell.chaobei.xyz/install.sh  |  bash

```
